 # Password Generator

 # Description
   Random Password Generator allows the user to quickly generate a password at their disposal with the choice of including lowercase/uppercase letters, numbers, and   special characters once accepted.
   
   # Technologies used
   - HTML
   - CSS
   - JAVASCRIPT
